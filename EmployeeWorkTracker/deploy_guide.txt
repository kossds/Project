ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ СИСТЕМЫ НА ЛЮБОМ КОМПЬЮТЕРЕ
====================================================================

ПРОСТОЙ СПОСОБ ЗАПУСКА:
1. Скопируйте всю папку проекта на целевой компьютер
2. Откройте командную строку/терминал в папке проекта
3. Выполните: python install.py
4. Запустите систему:
   - Windows: дважды кликните start_system.bat
   - Linux/Mac: ./start_system.sh
   - Или: python main.py

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Python 3.7 или выше
- Любая операционная система (Windows/Linux/Mac)
- 50 МБ свободного места на диске
- Браузер для доступа к системе

ДОСТУП К СИСТЕМЕ:
- Адрес: http://localhost:5000
- Админ: admin@company.com / admin123

ФАЙЛЫ ДЛЯ ПЕРЕНОСА:
Скопируйте ВСЕ файлы проекта:
- app.py, main.py, models.py, routes.py
- папка templates/ (все HTML файлы)
- папка static/ (CSS и JavaScript)
- install.py, setup.py
- start_system.bat, start_system.sh
- README.md

БАЗА ДАННЫХ:
- Использует SQLite (файл employee_tracking.db)
- Создается автоматически при первом запуске
- Не требует установки сервера БД
- Полностью портативна

УСТРАНЕНИЕ ПРОБЛЕМ:
1. "Python not found" - установите Python с python.org
2. "Permission denied" - запустите как администратор
3. "Port 5000 in use" - закройте другие приложения или измените порт в main.py
4. "Module not found" - запустите python install.py

АРХИВАЦИЯ ДЛЯ ПЕРЕНОСА:
Можете заархивировать всю папку проекта и перенести архив на любой компьютер.
После распаковки выполните setup.py или install.py для настройки.
